# Clinical Trial Adverse Event Analytics System (MySQL)


### Built for Data Analyst / Pharmacovigilance Roles
- 500+ realistic rows (Patients, AEs, Labs, ConMeds)
- MedDRA-coded Adverse Events
- Real Phase II/III trial structure
- 30+ advanced SQL queries (SAE rate, Hy’s Law, discontinuations, etc.)

### How to Run (2 minutes)
1. Run `1_Create_Database.sql`
2. Run `2_Insert_Data.sql`
3. Run `3_Advanced_Queries.sql`

### Key Queries Included
- Top 15 Most Common AEs
- Serious AE Rate by Country
- Grade 3–5 AEs by Treatment Arm
- Liver Toxicity Signal Detection 
- Drug-Related Discontinuations
